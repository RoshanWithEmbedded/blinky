# Changelog

All notable changes to Blinky will be documented here.

## [1.0.0] - 2025-01-XX

### Added
- Initial release
- 9 emotional states with physics-based animations
- OpenWeatherMap integration
- 5-page auto-rotating display
- Touch controls (tap, double-tap, long-press)
- WiFi configuration portal
- World clock with timezone support
- 3-day weather forecast
- Factory reset functionality
- Automatic config mode on WiFi failure

### Technical
- Spring-damper physics engine
- Software-based OLED initialization (no RST pin)
- NTP time synchronization
- Non-volatile preferences storage
- I2C communication at 100kHz

## [Unreleased]

### Planned
- MQTT support
- Additional mood states
- Mobile companion app
- Sound effects
